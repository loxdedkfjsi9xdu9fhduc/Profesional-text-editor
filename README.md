# PyEditor — профессиональный текстовый редактор для Python

Легковесный редактор кода с подсветкой синтаксиса, в стиле VS Code.

## Возможности

- **Подсветка синтаксиса Python** — ключевые слова, строки, комментарии, функции
- **Нумерация строк**
- **Несколько вкладок** — работа с несколькими файлами
- **Файловые операции** — New, Open, Save, Save As
- **Поиск и замена**
- **Запуск кода** — выполнение Python прямо из редактора
- **Тёмная тема** — комфортная работа для глаз
- **Автоотступы** — Tab и Enter с учётом текущего уровня

## Требования

- Python 3.7+
- Tkinter (обычно идёт с Python)

## Горячие клавиши

| Действие | Сочетание |
|----------|-----------|
| Новый файл | Ctrl+N |
| Открыть | Ctrl+O |
| Сохранить | Ctrl+S |
| Сохранить как | Ctrl+Shift+S |
| Найти | Ctrl+F |
| Заменить | Ctrl+H |
| Вырезать | Ctrl+X |
| Копировать | Ctrl+C |
| Вставить | Ctrl+V |
